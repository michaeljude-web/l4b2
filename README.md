## üìñ About

**Local Browser** turns any website into a clean, full-screen mobile app interface! This app hides browser tabs and the address bar, making your web content look like a native Android app. Perfect for showcasing websites, testing local web projects, or creating app-like browsing experiences without opening a standard browser.

## üöÄ How to Use

### Basic Usage

1. **Launch the App**
   - Open Local Browser on your device

2. **Enter URL or Path**
   - Type your URL in the input field
   - Supported formats:
     - Full URL: `https://example.com`
     - Local absolute path: `/storage/emulated/0/index.html`
     - Local relative path: `myproject/index.html`

3. **Browse Content**
   - Click "Enter" to load the page
   - Content displays in the full-screen iframe
